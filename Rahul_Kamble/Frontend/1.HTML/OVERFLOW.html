Overflow with auto scroll

You can use the overflow property when you want to have better control of the layout. The overflow property specifies what happens if content overflows an element's box.f you don't set the height of the box, it will grow as large as the content. But if you set a specific height or width of the box and the content inside cannot fit then what will happen. The CSS overflow property is used to overcome this problem. It specifies whether to clip content, render scroll bars, or just display content.
